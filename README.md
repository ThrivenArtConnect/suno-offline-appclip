# Suno Offline Prompt Viewer
Offlinefähige Web-App zur Darstellung von Suno-Prompts, Lyrics, Emojis und Stilbeschreibung. Entwickelt für NFC-Tags, QR-Codes und App Clips. Vollständig lauffähig im Browser – ohne Server oder Internet.
